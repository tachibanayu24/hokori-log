中国発の注目AIモデルが続々登場 最新AI動向まとめ

最近のAI界隈では、特に中国発の新しい大規模言語モデルや動画生成モデルの発表が相次いでいます。これらの新モデルを中心に、エージェント技術の進化、開発ツール、ローカルLLMの動向などをまとめて見ていきましょう。

中国発の新星たち：MiniMax、Hailuo、Moonshot

DeepSeekやQwenといった既存の強力な中国発AIモデルに続き、新たなプレイヤーが登場しています。

  • MiniMax-M1
    • 100万トークンの入力、8万トークンの出力に対応
    • 約456Bパラメータ（アクティブは約45.6B）のオープンウェイトLLM
    • 効率的な「Lightning Attention」とGRPOの派生「CISPO」を採用
    • 技術レポート (PDF)も公開
    • 長文コンテキスト処理と低コスト学習（約53万ドル）が特徴
  • Hailuo 02 (旧名 Kangaroo)
    • MiniMax社による動画生成モデル
    • ByteDanceのSeedanceモデルと同様、発表のみでウェイトやAPIは未公開
    • Artificial Analysisの動画アリーナでSeedance 1.0に次ぐ2位と高評価
      • GoogleのVeo 3 Previewを上回る
      • ただし生成に約20分かかり実用性はまだ低い
  • Moonshot AI Kimi-Dev-72B
    • コーディングに特化した72Bパラメータモデル
    • SWEBench VerifiedでDeepSeek R1を上回る性能
      • スコア60.4%を達成
    • 大規模強化学習パイプラインで実コードベースを自律的に修正
    • HuggingFaceでモデルが公開
    • 技術レポートは未公開

これらのモデルは、オープンソースコミュニティにとっても注目の的となりそうです。

AIエージェント技術の進化とセキュリティ

AIエージェントシステムとその開発も活発です。

  • マルチエージェントシステム
    • Anthropic社が本番グレードのマルチエージェント研究システム構築に関する知見を公開
      • 並列化に適したユースケースの選択
      • ツールインターフェース改善のためのエージェント活用（ツールテストエージェントでタスク完了時間40%削減）
      • 同期実行のボトルネック
    • LangChainやCognition Labsも同様の知見を共有
    • 一方で、逐次的な深掘りがない検索など「マルチエージェントの匂い」に懐疑的な声も
    • AnthropicのマルチエージェントシステムはClaude Opus 4をリード、Claude Sonnet 4をサブエージェントとし、単一Opus 4より90.2%高性能 (Anthropicブログ)
    • Claude SwarmはClaude CodeのMCP機能を活用し階層的専門家チームを形成 (GitHub)
  • AIプログラミングモデルの進化
    • 「マルチエージェント」や「マルチステージ」という区別は本質的でなくなりつつあるとの指摘
    • DSPyのようなフレームワークは、LLMを呼び出せる任意のプログラムで指示や重みを調整
  • エージェントのセキュリティ脆弱性
    • Andrej Karpathy氏がプロンプトインジェクション攻撃のリスクを指摘
      • Redditのような信頼されたサイト上の悪意のあるリンクでエージェントが操作される可能性
    • コロンビア大学の研究では、エージェントが100%のケースで罠にかかり機密情報を漏洩
  • 特化型エージェントの価値
    • 汎用チャットアシスタントに対し、特定タスクに特化した自動化エージェントが有効
    • LlamaIndexなどがプロコード視点でアプローチ
  • Sakana AIのALE-Agent
    • NP困難な最適化問題を解くために設計されたコーディングエージェント
    • AtCoderヒューリスティックコンペティションで1000人中21位と好成績
    • ALE-Benchデータセットとコードが公開
  • Model Context Protocol (MCP)
    • ツール使用とエージェント連携におけるMCPの重要性が高まる
    • GitHub MCP ServerやFastMCPなどのプロジェクトが登場
    • MicrosoftがData + AI SummitでLlamaIndex.TSとAzure AI Foundryを用いたAI旅行代理店をデモ

  • ChatGPTの言語スタイル

    • 英国の大学で約7000人の学生がAIを利用した不正行為で摘発されたと報道 (The Guardian記事)
    • 実際の不正件数はさらに多いと推測され、教育システム側の対策が急務
  • ドイツ企業のAI導入

    • 完全な職務代替には至らずとも、生産性向上や業務変革は既に大規模に進行
